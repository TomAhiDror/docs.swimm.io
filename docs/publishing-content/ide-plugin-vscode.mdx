# IDE Plugins - VS Code

### Step 1: Install
Get your Visual Studio Code extension <Link href="https://marketplace.visualstudio.com/items?itemName=Swimm.swimm">here</Link>


### Step 2: Login
:::note 
Our JetBrains plugin doesn't require a login. 
:::

**Login:** Once installed, you will see the Swimm panel in your IDE.
**Authorize:** In your browser you will be asked to authorize swimm

![Login](../../static/img/vscode-login.png)

### Step 3: Explore
![Overview](../../static/img/vscode-overview.png)

#### The Sidebar
Now that you have the extension installed and successfully logged in, 
you will see the Swimm sidebar. If you already have Swimm code-coupled documents you will see a list all of them.

![Authorize](../../static/img/vscode-panel.png)<br/>
**Clicking on an item will open the markdown document**.

:::tip
Get the best out of your md files with markdown preview extensions in the Visual Studio marketplace
:::

In the **"Current File"** section you will see documents associated with the code file you are working on.

#### Code Hints

Here's how Swimm code hints relate between the IDE and in the App. <br/>
Snippets have a section to capture a description. <br/>
The description will be available in the context of your code.
![Overview](../../static/img/code-comments.png)

Hover the Swimm icon to see more about the documentation.<br/>
You will see the Documenent name and some metadata associated with the document which is related to this particular piece of code.<br/>
![Overview](../../static/img/icon-hover.png)

:::tip
Looking at the creator and last contributor is a great way to find out who has the knowledge related to the code.
:::
