# Swimm 0.7.3 Has Been Released!

We're pleased to announce the availability of **Swimm 0.7.3**<SwimmMoji text="release"/>

<SwimmReleaseBlogPost version="0.7.3" />

<SwimmReleaseVideo version="0.7.3" />

 
## Improvements:
 
 - Performance of viewing Swimm Docs has improved.
 - Performance of "Commit to branch" action has improved.
 - Review of outdated/auto-synced smart paths now possible.
 - Support for .swm files is now deprecated.


## Bug Fixes:

 - IntelliJ Plugin: Fixed issue where in some cases the lifebuoy was not displayed.
 - IntelliJ Plugin: Fixed issue where switching between branches led to an error.

If you have any questions or issues, please reach out on our <SwimmLink target="slack" />.
