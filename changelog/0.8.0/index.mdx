# Swimm 0.8.0 Has Been Released!

We're pleased to announce the availability of **Swimm 0.8.0**<SwimmMoji text="release"/>

<SwimmReleaseBlogPost version="0.8.0" />

<SwimmReleaseVideo version="0.8.0" />

## New Features:
 - You can use Google Sign-In to sign up or login to Swimm
 - Introducing Swimm Generated Docs! 
Swimm will automatically detect the important parts of any repo within seconds, 
and generate full code-coupled documents about them for use right after connecting the repo to Swimm. 
Currently, only Python, C++, Java, C# codebase are supported.

## Bug Fixes:
 - You can now select Smart token from a typescript-based repository

If you have any questions or issues, please reach out on our <SwimmLink target="slack" />.
