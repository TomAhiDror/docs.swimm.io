# Swimm 0.9.9.12 Has Been Released!

We're pleased to announce the availability of **Swimm 0.9.9.12**<SwimmMoji text="release"/>

<SwimmReleaseBlogPost version="0.9.9.12" />

<SwimmReleaseVideo version="0.9.9.12" /> 

## Improvements:
- **Unified Workspace Settings** - The Edit Workspace, Billing & Plans, and Invite pages are now merged into a single workspace settings area, available in the bottom-left corner. 
- **Viewing and Editing Docs** - Performance is smoother when switching between viewing and editing docs.
- **Reverting Draft Changes from Batch Commits** - A committed change can now be discarded by reverting draft changes from the batch commit modal located in the top right corner of the repository page.
- **Editor Diagram Examples** -  Select a diagram template (e.g., sequence diagram, flow chart, etc..) and start editing the code.
- **Editing Playlists** - When you edit a Playlist, if you remove a blank doc from the Playlist, the doc will be discarded, and the documentation step will be removed from the Playlist.
- **Streamlined Searching** - Improved searching with the search modal in repos and not only in workspaces.
- **Deleting Documentation** - You can now delete documentation from a search result.
- **Home Page Doc Creation** - You can now create a doc from the home page.

## Bug Fixes:
- Fixed: issue using templates in a Playlist.
